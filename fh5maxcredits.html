Max Credits With AIO

1. Make sure you have done the sandstorm photo mission first

2. In AIO stat editor change TotalWinnings to 999999999, TotalCredits to 0, Credits to 0

3. Open a wheelspin

4. Enjoy max credits

OR

1. In AIO stat editor, add 5 to the value of TotalCredits and Credits

2. Send the values

3. Open a wheelspin

4. Enjoy max credits
