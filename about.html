About AIO

Forza Mods AIO is a free Forza Horizon 4, and now Forza Horzion 5 mod tool that aims to bring cool, new and unique features, all at NO cost!

How do add cars? Where is the car IDs list?

Using the AIO which supports both FH4 and FH5. You no longer need a car ID list.

What platforms does the AIO support?

It is predominantly a microsoft store tool as both of the devs use this version but features can and will be ported to the steam version in the future. (cracked versions of the game EG empress are not officially supported but the tool may still work)

Is the AIO a virus?

Due to the nature of obfuscated cheats it may sometimes be flagged by an antivirus. I can assure you it is a false positive (you can even check yourself by viewing the network traffic of the tool)

I cant download/launch the AIO

Make sure your antivirus is off and/or you have microsoft .net framework installed which can be downloaded from microsoft.

How do I get credits?

Add cars for free using the AIO and sell them on the auction house. Adding credits directly is pretty risky so using this method will give you "clean" money that is accounted for in game.
