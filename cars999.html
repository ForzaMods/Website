Cars with 999 Engine

Go to car collection and select the formula drift viper

Open 999 Engine, and select forza

Set the search value to 3003 and click first scan

Move your mouse around the forza window and comeback to 999 Engine

Click next scan 2-3, and select all values except green (dont look for the green if you dont see it)

Change all values of the selected addresses to the id of the car you want

Now buy the car with y button

If your game crashes when changing values try again

This part prevents the crash after changing values

Rightclick the selected addresses

Click on the third/fourth option (change value to first/last), select first and click ok
