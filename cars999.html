Cars with 999 Engine

- Set the search value to 3003 (FD Viper ID)

- Scan for the id

- When here go one brand up and one brand down

- When done hit the next scan button

- Select all of the values except the green one

- Search for the id of the car you want and replace it in the selected values

Prevent crashing

- Put the id of the car you bought into the value box and hit next scan

- Change the addresses value to 3003

- Now you can exit without crash
