Quick model swap and fix textures on swapped parts

Skip to #5 for texture only.

1: Go to \media\Cars\ and choose both the car you want to mod and the one you are swapping the model/texture from

2: Copy both into a new workings folder and extract zips into seperate folders (modded and donor)

3: Open the donor car folder, locate the part you want to swap onto another car and copy it (wing_a.modelbin for example)

4: Open the modded car folder, locate the part you want to replace with the donor model, replace it with the copied donor .modelbin.

5: Open the donor car folder, locate the donor .carbin file in folder root and open into a hex editor (NIS_SilviaSpecR_00.carbin for example)

6: Search for the modelbin string you swapped such as Wing_a.modelbin

7: Count 0x46 (70 bytes) after the modelbin string to see the string , hex 3C 72 6F 6F 74 3E

8: Select and copy from the start of to the end of the .swatchbin, right before hex FF FF FF FF FF FF FF FF

9: Open the modded car folder and repeat from step 4, then copy the donor selected bytes over the modded selected bytes then hit save.

10: All texture errors should now be fixed on the model swapped part.
