The method itself

Go to the car mastery of the car you have chosen

In the value box set the value of the cr skillpoint

Set the scan type to float, truncated. then click first scan

Add .00 at the end of the value box and click next scan 4-5 times

Select all addresses and then change them to whatever you want them to be

When changed unlock the cr skillpoint
