Smoked/Clear tails tutorial

Names may may differ for the material entries

Open the modelbins with the name of: glass_RTL and glass_LTL

Run D3's template

Im using the 2018 Mustang GT so i open the only material entry called gls_colored.materialbin

Now open it and scroll down to the RGB values

For smoked tails do 0 for all values

For clear tails do 1 for all values

Save the first modelbin and repeat steps for the other
