Saveswap for Steam

Take note of the xuid of the target save and your own profile save (XUID Grabber)

Log out of the game on the title screen and realaunch the game

Press log in

Change the result to the XUID of the save you want

Press lets go and wait on the nickname selection screen

Check your save folder for a new XUID folder (if it exists close your game)

Replace the .profiledata in that folder with the save you want

Repeat steps 1 through 6 and it should take you in game with the new save

You will load in on a "fake" xbox account with that persons Gamertag

Search for every instance of the target XUID and replace it with yours

Switch cars to force a save and close the game

There will be a second profiledata file in your original XUID folder with the same file name as the one in the new XUID folder

Replace your original save with this save and relaunch the game
