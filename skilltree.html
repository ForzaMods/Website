Getting the required address

You are going to use the XP Guide or the Credits Guide to get the list of addresses you need

When you have the list you need, change half of the addresses to whatever you want, exit car mastery, comeback

If the skillpoint value changed (forza car mastery):

Delete half of the addresses you didnt change

Change the half you got

Exit car mastery, if changed repeat

If the skillpoint value didnt change (forza car mastery):

Delete the half you changed

Change the half you got

Exit car mastery, if changed do the "skillpoint changed" part. If not, repeat

Setting up 999 Engine for scanning

We want to modify the start and stop scanning options, with the address we have

Start and stop has 16 letters, we leave the first five 0's and modify the rest

The address for example 1C6F7D35190 has 11 letters, we take the first four, paste it after the first five 0's and then copy this to the stop section

Now in our stop section, change the 0's at the end to F's

Example of how it would look: Start: 000001C6F0000000, Stop: 000001C6FFFFFFFF. Remember this wont be the same

Scanning

Lets say you want to modify the drift percentage,

In the value box you type in the skillpoint percentage, then first scan

Now select scan type: unchanged value and click next scan couple times

At the end of the value box add .0 and click next scan one time to filter out most of the useless results

Select and change all addresses to whatever you want, then if everything is done correctly the drift percentage should change after unlocking the skillpoint
