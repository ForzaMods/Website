Setting up 999 Engine

So 999 Engine, has memory scanning options

What were intrested in is Start, Stop

The value box of start and stop has 16 spaces for numbers and letters

So what we want to do is modify that memory searching to our likings

Getting the right scanning options

So in the start we leave the first five 0's and modify the rest

Our address for example 1C6F7D35190 has 11 letters, so we get the first five of them and paste it after the first five 0's and leave the rest

Example of how it should look: Start Scan: 000001C6F7000000

Now for the stop scan we do copy paste the Start Scan, but replace the six 0's at that end with six F's

Example of how it should look: Stop Scan: 000001C6F7FFFFFF

Remember the address wont be the same

Scanning

Here we start off with setting the value of the skillpoint you want to modify

Lets say we want to modify a wheelspin value (the wheelspin value needs to give you atleast two of them)

So in the value box you type in 2, then next first scan

You should get something from 40-150 addresses, then add .0000 at the end of your value box and next scan

Now select Scan Type unchanged value and next scan couple times

Modify all of the values to your liking, then exit the car mastery and comeback to it

Now your wheelspin value should be number you wanted it to be
