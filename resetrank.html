FH5 Reset Rank

1. Get total XP value from stat editor.

2. do 4294967295 minus total XP in a calculator.

3. halve it, and give yourself that much xp.

4. add 5 to value you got in #3 and give yourself that amount of xp

5. you should now be at level 1

If not working, first try:

1. set scan type to unknown and search in Draff Engine.

2. give yourself 6942069 xp once

3. set scan type to "increased value by," put 6942069 in the box and search.

4. add the first one (otherwise the most frequent value) from the list to the address table at the bottom (double click it) - this is your total xp.

5. do 4294967295 minus total xp in a calculator.

6. halve it, and give yourself that much xp.

7. add 5 to value you got in #6 and give yourself that amount of xp

8. you should now be at level 1
