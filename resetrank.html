FH5 Reset Rank

AIO Method (works when AIO is up)

1. Get total XP value from stat editor.

2. do 4294967295 minus total XP in a calculator.

3. halve it, and give yourself that much xp.

4. add 5 to value you got in #3 and give yourself that amount of xp

5. you should now be at level 1

999 Engine Method

you need to get 3 apollos for this and 2 instances of 999 engine

1. set scan type to unknown and scan (while in freeroam) in the first window of 999 engine

2. give yourself 6942069 xp in the second window of 999 engine

3. set scan type to increased by and in the value box set "6942069" and click next scan in the first window of 999 engine

4. add the first address (total xp) that popped up to the table in the first window of 999 engine

5. do 4294967295 minus total xp in calculator

6a. get in the 2nd apollo

6b. halve it, subtract 200 and give yourself that much xp

7a. get in the 3rd apollo

7b. add 5 to the value you got in step 6b and give yourself that much xp
