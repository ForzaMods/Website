Recommendations

Any car that has the: 7000, 9000, 15000 xp skillpoint, because they get the lowest ammount of addresses

Must haves

Atleast one skillpoint

‎

The method itself

Go to the car mastery of the car you have chosen

In the value box set the value of the xp skillpoint

Set scan type to float, truncated and click first scan

Add .00 at the end of the value box, then hit next scan 2-3 times

Select all addresses and change them to whatever you want

After changing the value of all addresses you can unlock the skillpoint
